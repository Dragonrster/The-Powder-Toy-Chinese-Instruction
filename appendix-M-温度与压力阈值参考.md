```
ICE ──273K──▶ WATR ──373K──▶ WTRV ──冷凝──▶ WATR
 ▲                                        │
 └────────────── 冷冻 ────────────────────┘

```
METL ──1273K──▶ LAVA(METL) ──冷却──▶ METL
                                │
                高压>50 ──▶ BRMT或CNCT

                                                      │

## 温度与压力联动效应

### 高温+高压组合

| 组合条件 | 效果 | 说明 |
|---------|------|------|
| T>5000K + P>100 | LAVA→URAN/PLUT/TUNG | 极端条件核材料生成 |
| T>9973K + P>250 | O2核裂变 | 极端核反应 |
| T>573K + P<10 | 燃烧扩散加速 | 低气压促进燃烧 |
| T>373K + P>5 | WATR不蒸发(高压抑制) | 高压保持液态 |

### 低温+负压组合

| 组合条件 | 效果 | 说明 |
|---------|------|------|
| T<273K + P<0 | 水直接升华(固→气) | 跳过了液态阶段 |
| T<77K + 负压 | LN2快速蒸发 | 极低温真空 |
| T<63K + 负压 | NICE升华 | 氮冰直接成气 |

## 常见温度控制方法

| 方法 | 元素 | 效果 |
|------|------|------|
| 加热 | HEAC(加热器) | 持续升温至目标温度 |
| 冷却 | COOL(冷却器) | 持续降温至目标温度 |
| 快速加热 | FIRE | 573K以上直接点燃 |
| 快速冷却 | LN2 | 直接降至77K |
| 精确控制 | HEAC+COOL+Lua | 通过TICK事件PID温度控制 |
| 环境恒温 | 环境温度设置 | 全局基线温度 |
| 无热交换 | PROP_NOAMBHEAT | 禁用与环境的热传导 |
