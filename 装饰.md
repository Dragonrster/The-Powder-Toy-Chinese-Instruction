# 装饰 (Decoration)

装饰工具用于在存档上绘制颜色和纹理，不影响物理模拟。共 **7** 种工具，使用 DECO 颜色混合模式。

## 装饰工具速查

| 工具 | DECO模式 | 操作 | 效果 |
|------|----------|------|------|
| SET | 0 | 直接赋值 | dcolour = 画笔颜色 |
| CLR | 1 | 清空 | dcolour = 0（恢复元素原生色） |
| ADD | 2 | 相加 | dcolour = dcolour + 画笔颜色（溢出钳制） |
| SUB | 3 | 相减 | dcolour = dcolour - 画笔颜色（下限钳制） |
| MUL | 4 | 相乘 | dcolour = dcolour * 画笔颜色 / 255（变暗/染色） |
| DIV | 5 | 相除 | dcolour = dcolour * 255 / 画笔颜色（变亮） |
| SMDG | 6 | 涂抹 | 周围粒子颜色混合后赋值（模拟手指涂抹） |

## 什么是装饰层

TPT 的装饰系统独立于粒子物理模拟。每个粒子除了物理属性外，还有一个 `dcolour`（装饰颜色）字段——32位ARGB值，存储在粒子的图形层中。装饰工具用于修改这个字段，完全不影响粒子的物理行为——不会改变温度、压力、运动、反应等模拟结果。

### 技术细节

- **dcolour 格式**：32 位 ARGB（Alpha-Red-Green-Blue），每通道 8 位（0~255）
- **默认值**：0 表示"使用元素原生颜色渲染"
- **DECO 模式**：每个装饰工具对应一个混合模式编号（0~6），存储在粒子的装饰标记中
- **渲染优先级**：dcolour 优先于元素原始颜色，除非在设置中关闭装饰层显示（Ctrl+B 切换）
- **存档兼容**：dcolour 随存档保存和分享

## 绘制工具 (SET)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_DECOR_SET |
| DECO模式 | 0 |
| 描述 | 直接绘制装饰颜色，无混合 |

### 核心行为
- 将粒子的 dcolour 直接设置为所选颜色
- 无任何混合/叠加效果
- 覆盖已有的装饰颜色

### 应用场景
- 基础颜色绘制
- 标记区域
- 彩色线条和形状

---

## 清除工具 (CLR)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_DECOR_CLR |
| DECO模式 | 1 |
| 描述 | 清除任何设置的装饰 |

### 核心行为
- 将粒子的 dcolour 重置为默认值（0 或 0xFF000000，使用元素默认颜色）
- 擦除所有装饰效果

### 应用场景
- 装饰橡皮擦
- 重置颜色

---

## 相加混合 (ADD)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_DECOR_ADD |
| DECO模式 | 2 |
| 描述 | 颜色混合：相加 |

### 核心行为
- 将所选颜色与粒子现有 dcolour 相加
- 结果可能超过最大值并溢出（产生更亮的效果）
- 多通道叠加会产生逐渐增亮的颜色

### 应用场景
- 创建发光效果
- 颜色叠加
- 渐变增亮

---

## 相减混合 (SUB)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_DECOR_SUB |
| DECO模式 | 3 |
| 描述 | 颜色混合：相减 |

### 核心行为
- 从粒子现有 dcolour 中减去所选颜色
- 结果钳制在 [0, 255] 范围
- 多通道叠加产生逐渐变暗的颜色

### 应用场景
- 创建阴影效果
- 颜色减弱
- 渐变变暗

---

## 相乘混合 (MUL)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_DECOR_MUL |
| DECO模式 | 4 |
| 描述 | 颜色混合：相乘 |

### 核心行为
- 将所选颜色与粒子现有 dcolour 相乘（归一化后）
- 产生非线性混合效果
- 通常用于加深或染色

### 应用场景
- 色调调整
- 深度着色
- 颜色过滤

---

## 相除混合 (DIV)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_DECOR_DIV |
| DECO模式 | 5 |
| 描述 | 颜色混合：相除 |

### 核心行为
- 将粒子现有 dcolour 除以所选颜色（归一化后）
- 产生增亮的非线性效果
- 与 MUL 相反

### 应用场景
- 高光效果
- 颜色淡化
- 特殊色彩处理

---

## 涂抹工具 (SMDG)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_DECOR_SMDG |
| DECO模式 | 6 |
| 描述 | 混合周围装饰 |

### 核心行为
- 将周围粒子的装饰颜色混合并涂抹
- 模拟手指涂抹颜料的效果
- 产生平滑的颜色过渡

### 应用场景
- 颜色平滑过渡
- 艺术效果
- 纹理混合
