# 附录K：SEED/PLNT种子遗传系统

## SEED生命周期（完整流程）

```
                  ┌──────────────────────────────┐
                  │                              │
                  ▼                              │
[SEED种子] ──吸水──▶ [发芽阶段] ──life>200──▶ [PLNT树木模式]
    │                  │                              │
    │                  │ 缺水/损伤                     │ 生长完成
    ▼                  ▼                              ▼
[死亡]          [SEED枯萎]                    [PLNT普通模式]
                                                   │
                                                   ▼
                                          吸收WATR→分裂繁殖
```

---

## 各阶段详解

### 阶段1：吸水(Water Absorption)
- **吸收对象**：WATR(水)、DEUT(重水)、DSTW(蒸馏水)、CBNW(碳酸水)
- **吸水效果**：每接触一次水粒子，`tmp`值增加（表示水分含量）
- **吸水范围**：SEED粒子周围接触即可吸收
- **吸水速率**：接触即吸收，无延迟

### 阶段2：损伤机制
- **SLTW盐水接触**：接触SLTW会减少水分(tmp减少)
- **高温损伤**：>373K(100℃)时水分加速流失
- **高压损伤**：>5压力时生长受阻

### 阶段3：发芽条件检查
需要**同时满足**两个条件：
1. **水分充足**：`tmp >= 4`（吸收了足够的水）
2. **土壤条件**：下方恰好是SAND(沙子)
3. **空中条件**：上方为空(NONE)
4. 满足条件后`life`逐帧累积

### 阶段4：树木生长
- `life > 200`时转化为PLNT进入树木生长模式
- `life`达到200约需200帧（正常速度约3.3秒）

### 阶段5：高温熔化
- 极高温度：>319K(46℃)→MWAX熔化
- 超高温：>573K(300℃)→直接燃烧

## PLNT树木生长（基因编码系统）

### 基因存储结构

PLNT在树木模式下使用`tmp`~`tmp4`存储基因信息：

| 基因字段 | 存储位置 | 编码内容 | 影响 |
|---------|---------|---------|------|
| 颜色基因 | tmp | 高低4位编码主色和次色 | WOOD和叶子颜色 |
| 分支角度 | tmp2 | 低8位=角度, 高8位=偏转 | 分支生长方向 |
| 分支长度 | tmp2 | 中8位 | 每个分支段长度 |
| 生长方向 | tmp3 | 方向编码(0=上,1=右,2=下,3=左) | 主干走向 |
| 分支频率 | tmp3 | 每N段分叉一次 | 分支密度 |
| 特殊形态 | tmp4 | 位编码: 0=正常,1=螺旋,2=弯曲 | 树木形态类型 |
| 种子色 | dcolour | 射出SEED的颜色继承 | 后代初始颜色 |

### 生长阶段

```
阶段A: 茎秆生长期
[PLNT] ──生成──▶ [WOOD(木)] ──> [GOO(藤茎)]
颜色由tmp基因决定

阶段B: 分支期
[PLNT顶端] ──按角度──▶ [分支1(WOOD+叶子)]
                     ──▶ [分支2(WOOD+叶子)]
叶子颜色=次色基因，茎杆颜色=主色基因

阶段C: 繁殖期
[PLNT顶端] ──喷射──▶ [SEED种子]
SEED携带50%父本基因+50%变异

阶段D: 退化
树木本身转为普通PLNT模式
吸收WATR后继续繁殖(无树木形态)
```

## 基因详细说明

### 颜色基因编码表

| 基因值 | 主色(茎杆) | 次色(叶子) | 树型 |
|--------|-----------|-----------|------|
| 0 | 棕色(经典) | 深绿 | 标准落叶树 |
| 1 | 深棕 | 浅绿 | 常绿树 |
| 2 | 灰棕 | 黄绿 | 热带树 |
| 3 | 红棕 | 红色 | 红叶枫树 |
| 4 | 白灰 | 蓝绿 | 白桦树 |
| 5 | 黑棕 | 紫绿 | 暗影树 |
| 6 | 金棕 | 黄 | 秋色树 |
| 7 | 绿棕 | 橙 | 奇异树 |
| 8-15 | 混合色 | 渐变 | 随机变异体 |

### 颜色遗传规律

```
父本颜色A × 母本颜色B
    │
    ▼
50%概率：A（完全遗传父本）
30%概率：B（完全遗传母本）
15%概率：(A+B)/2（混合中间色）
4%概率： 随机新颜色（突变）
1%概率： 极端颜色（跳跃突变）
```

### 分支基因说明

| 基因参数 | 范围 | 默认值 | 说明 |
|---------|------|--------|------|
| 主干方向 | 0-3 | 0(上) | 0=上, 1=右, 2=下, 3=左 |
| 分支角度 | 0-180° | 45° | 0=直线, 45°=标准分支, 90°=水平分支 |
| 分支长度 | 1-50格 | 8格 | 每个分支段的最大长度 |
| 分支频率 | 1-20 | 5 | 每N格分叉一次 |
| 分支层数 | 1-10 | 3 | 最多几层分叉 |
| 螺旋度 | 0-360 | 0 | 0=正常, >0=螺旋生长 |
| 弯曲系数 | 0-100 | 0 | 0=笔直, >0=弯曲生长 |

### 生长方向与光照优化
树木会微调生长方向以优化"光照"（基于上方空位数）：
- 上方连续空位多→向上生长加速
- 上方有阻挡→侧向分支增加

## 基因杂交

两颗含水的SEED相邻接触时发生基因杂交：

```
[SEED A] ◄══接触══► [SEED B]
(tmp>=4)            (tmp>=4)
    │                  │
    └─────┬──────┬─────┘
          │      │
   颜色基因交叉   分支基因混合
          │      │
          ▼      ▼
    新SEED产生（在A位置或B位置）
```

### 杂交规则
- **颜色基因**：父本+母本随机交叉遗传
  - 70%: 父本主色+母本次色
  - 20%: 母本主色+父本次色
  - 10%: 随机组合
- **分支基因**：父本+母本随机混合
  - 每项50%概率选父本或母本的值
- **杂交产出**：一次杂交产出一颗SEED
- **杂交冷却**：杂交后有短暂冷却期(约30帧)不能再次杂交
- **后代变异**：后代可能有新的颜色/形态变异（约5%概率）

## 辐射诱变育种指南

### NEUT中子轰击
- **概率**：每次50%(1/2)概率触发
- **效果**：随机翻转一个基因位
- **可影响**：颜色基因、分支角度、分支长度、形态类型
- **安全性**：可能杀死SEED（低概率，约1/10）

### 人工诱变育种流程
```
1. 准备基础SEED(选择优良性状亲本)
2. 放置低剂量NEUT源(控制距离=控制剂量)
3. 观察SEED变化(颜色/形态)
4. 选择目标变异SEED移至培养区
5. 重复诱变或与其他品种杂交
6. 得到稳定遗传的新品种
```

### PROT质子接触
- **概率**：50%(1/2)
- **效果**：交换两个随机基因段（不引入新基因，仅重组）
- **用途**：基因重组育种（不同于NEUT的随机突变）

### 辐射剂量指南

| NEUT距离(格) | 有效剂量 | 突变概率 | 致死率 | 适用场景 |
|-------------|---------|---------|--------|---------|
| 1-2 | 极高 | ~80% | ~30% | 极端突变 |
| 3-5 | 高 | ~50% | ~15% | 常规诱变 |
| 6-10 | 中 | ~25% | ~5% | 温和育种 |
| 11-20 | 低 | ~10% | ~1% | 精细调整 |
| >20 | 极低 | ~2% | ~0.1% | 长期慢性突变 |

## 普通模式PLNT

非树木模式的PLNT行为：

### 繁殖行为
- **吸收WATR**：接触水后吸收，积累水分
- **分裂繁殖**：水分充足时1/50概率产生新PLNT
- **自我维持**：寿命有限但通过繁殖延续

### 光合作用
- **条件**：life==2时（特定生长阶段）
- **效果**：周围产O2(氧气)
- **范围**：3x3邻域
- **周期**：每隔若干帧产生一次

### 环境互动

| 触发条件 | 效果 | 概率 |
|---------|------|------|
| 邻接WOOD | 产生VINE(藤蔓) | 偏高 |
| 接触SLTW(盐水) | 杀死PLNT | 1/40每帧 |
| 接触NEUT(中子) | 转为WOOD(木质化) | 1/20每帧 |
| 吸收SMKE(烟) | 延长寿命 | 每吸收+1寿命 |
| 吸收CO2 | 延长寿命 | 每吸收+2寿命 |
| STKM(火柴人)吃 | STKM+5生命值 | 必定(1次) |
| 高温(>573K) | 转化为FIRE燃烧 | 必定 |

### PLNT生命周期
```
[新PLNT] ──生长──▶ [成熟PLNT] ──繁殖──▶ [新PLNT(s)]
   │                    │
   │                    │ SLTW/高温/老死
   ▼                    ▼
[死亡→NONE]      [死亡→NONE或燃烧→FIRE]
```

## 树木培育实战指南

### 目标1：培育红色高树
```
步骤：
1. 基础SEED(品种0) + 大量WATR → 正常树木
2. NEUT诱变(距离3-5格) → 颜色变异
3. 选出红色变异SEED(品种3)
4. 红色SEED + 高分支SEED杂交 → 红色高树
5. 多代选育 → 稳定品种
```

### 目标2：培育螺旋盆景
```
步骤：
1. 基础SEED + 低剂量NEUT(距离10) → 形态变异
2. SEED.tmp4检查 → 寻找螺旋标记(位1)
3. 选出螺旋SEED + 短分支SEED杂交
4. 逐代缩短分支长度(选tmp2中位值最小的后代)
5. 最终得到小型螺旋盆景树
```

### 目标3：建造自动林木场
```
配置：
[水源WATR] → [SEED阵列] → [NEUT诱变区] → [选育区] → [成品区]
  ↑                                              │
  └──────────────── 自动循环灌溉 ────────────────┘

Lua脚本自动化：
- TICK事件检测SEED生长状态
- 自动收集成熟SEED
- 自动分类不同品种
```

## Lua基因操作指南

```lua
-- 读取SEED基因
local seedId = sim.partID(100, 100)
if seedId then
    local colorGene = sim.partProperty(seedId, 9)   -- tmp
    local branchGene = sim.partProperty(seedId, 10) -- tmp2
    local growthGene = sim.partProperty(seedId, 11) -- tmp3
    local morphGene = sim.partProperty(seedId, 12)  -- tmp4
    
    tpt.log(string.format("颜色:%d 分支:%d 生长:%d 形态:%d",
        colorGene, branchGene, growthGene, morphGene))
end

-- 手动设置基因（育种类）
function setSeedGenes(id, color, branch, growth, morph)
    sim.partProperty(id, 9, color)
    sim.partProperty(id, 10, branch)
    sim.partProperty(id, 11, growth)
    sim.partProperty(id, 12, morph)
end

-- 杂交两个SEED（手动触发）
function crossBreed(id1, id2)
    local c1 = sim.partProperty(id1, 9)
    local c2 = sim.partProperty(id2, 9)
    -- 随机组合基因
    local newColor = math.random() < 0.5 and c1 or c2
    local newBranch = math.random() < 0.5 and
        sim.partProperty(id1, 10) or sim.partProperty(id2, 10)
    -- 在id1位置产生新SEED
    local x, y = sim.partPosition(id1)
    local newId = sim.partCreate(-1, x, y, "SEED")
    if newId > 0 then
        setSeedGenes(newId, newColor, newBranch, 0, 0)
    end
end
```
