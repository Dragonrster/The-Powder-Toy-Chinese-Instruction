## 气压工具 (AIR)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_TOOL_AIR |
| 颜色 | 白色 (0xFFFFFF) |
| 描述 | 增加目标区域内的气压 |

### 核心行为
- 点击或拖拽时增加指定位置的气压值
- 气压增加量和范围由工具设置决定
- 气压变化会影响粒子的运动和行为

### 应用场景
- 创建高压区域
- 触发压力敏感元素的状态转换
- 驱动气流和粒子运动

## 属性笔 (PROP)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_TOOL_PROP |
| 颜色 | 多色（根据所选属性变化） |
| 描述 | 编辑粒子属性（温度、生命值、Ctype、Tmp 等） |

### 核心行为
- 右键点击任意粒子打开属性编辑窗口
- 下拉菜单选择要修改的属性字段（temp/life/ctype/tmp/tmp2/dcolour 等）
- 在文本框中输入新值并确认
- 点击后该粒子立即获得新属性值
- 支持批量操作：拖拽时范围内所有粒子统一设置

### 可编辑属性
| 字段 | 说明 |
|------|------|
| type | 粒子类型（元素种类） |
| life | 生命值 |
| ctype | 关联元素类型 |
| temp | 温度（开尔文） |
| tmp | 临时参数1 |
| tmp2 | 临时参数2 |
| dcolour | 装饰颜色（ARGB） |
| vx / vy | 速度分量 |
| flags | 粒子标志位 |

### 应用场景
- 精确调试元素行为参数
- 手动触发特定状态转换
- 修复存档中的错误粒子状态
- 快速测试元素在不同参数下的行为

## 文字标签 (SIGN)

| 属性 | 值 |
|------|-----|
| 内部标识 | DEFAULT_TOOL_SIGN |
| 颜色 | 白色文字框 |
| 描述 | 在画布上放置可编辑的文字标签 |

### 核心行为
- 点击画布上的空白区域放置文字标签
- 自动打开编辑窗口：输入文字、选择对齐方式
- 标签显示在 HUD 层，不影响物理模拟
- 支持特殊格式字段：
  - `{p}` — 显示当前位置压力值
  - `{t}` — 显示当前位置温度
  - `{c:元素名}` — 显示指定元素的粒子计数
  - `{a}` — 显示环境温度
- 标签可拖拽移动位置
- 右键已有标签可重新编辑或删除
- 标签存储在存档中，随存档保存和分享

### 对齐方式
| 模式 | 效果 |
|------|------|
| 左对齐 | 文字从标签点向右延伸 |
| 居中 | 文字以标签点为中心 |
| 右对齐 | 文字从标签点向左延伸 |

### 应用场景
- 存档注释和说明文档
- 实时数据仪表盘（配合 {p}/{t} 显示压力温度）
- 交互式教程指引
- 多人存档中的沟通标记
