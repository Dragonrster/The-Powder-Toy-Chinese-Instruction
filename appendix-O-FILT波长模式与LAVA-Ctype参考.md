# 附录O：FILT波长模式与LAVA Ctype参考

## FILT滤镜波长模式（完整版）

FILT修改穿过光子的波长（存储在Ctype中）。

### 所有模式详解

| Tmp | 模式 | 效果 | 公式 | 用途 |
|-----|------|------|------|------|
| 0 | 直接赋值 | 光子波长=FILT.ctype | `photon.wl = filt.ctype` | 固定颜色输出 |
| 1 | AND(位与) | 光子波长&FILT.ctype | `photon.wl &= filt.ctype` | 过滤特定波长 |
| 2 | OR(位或) | 光子波长\|FILT.ctype | `photon.wl \|= filt.ctype` | 添加波长分量 |
| 3 | 减法 | 光子波长-FILT.ctype | `photon.wl -= filt.ctype` | 去除波长 |
| 4 | 红移 | 根据温度向红色偏移 | `photon.wl >>= 1`(每级) | 模拟宇宙红移 |
| 5 | 蓝移 | 根据温度向蓝色偏移 | `photon.wl <<= 1`(每级) | 模拟蓝移 |
| 6 | 透传 | 不修改波长 | `不变` | 透明的FILT |
| 7 | XOR(位异或) | 光子波长^FILT.ctype | `photon.wl ^= filt.ctype` | 翻转特定波长位 |
| 8 | NOT(位取反) | 位取反(低位限定) | `photon.wl = ~photon.wl & 0x3FFFFFF` | 颜色反转 |
| 9 | 随机微调 | 微小随机偏移 | `photon.wl += random(-R,R)` | 模拟大气扰动 |
| 10 | 红移v2 | 基于filtWl最低位的红移 | 最低位决定位移方向 | 精细红移控制 |
| 11 | 蓝移v2 | 基于filtWl最低位的蓝移 | 最低位决定位移方向 | 精细蓝移控制 |

### 红移/蓝移的温差效应

红移和蓝移的幅度由FILT的温度控制：

| 温度范围(K) | 红移幅度 | 蓝移幅度 | 视觉效果 |
|------------|---------|---------|---------|
| <173K(-100℃) | 4位移位 | 4位移位 | 极强色移 |
| 173-273K | 3位移位 | 3位移位 | 强色移 |
| 273-373K(0-100℃) | 2位移位 | 2位移位 | 正常色移 |
| 373-573K(100-300℃) | 1位移位 | 1位移位 | 弱色移 |
| >573K(>300℃) | 0位移位 | 0位移位 | 几乎无效果 |

**温度公式**：
- 红移位移量 = `max(0, 4 - floor((temp - 173.15) / 100))`
- 蓝移位移量 = `max(0, 4 - floor((temp - 173.15) / 100))`

## 光子波长编码机制

### 波长数据结构

光子的波长存储在`Ctype`字段（底层为c, 32位值）：

```
位布局 (32位):
┌────────────┬──────────────────────────────────────────────────┐
│ 31 30...26 │ 25 24 │ 23 22 │ 21 20 │ ... │ 3 2 │ 1 0 │
│ 保留/标志   │ 组11  │ 组10  │ 组9   │ ... │ 组1 │ 组0 │
│            │ R  G  │ R  G  │ R  G  │     │ R G │ R G │
└────────────┴──────────────────────────────────────────────────┘

共12组(组0~组11)，每组2位编码一个RGB分量：
├── 00 = 无光(Off)
├── 01 = 红光(Red)
├── 10 = 绿色(Green)
├── 11 = 蓝色(Blue)
└── 低24位用于颜色编码(12组×2位)
```

### 波长编码表

| 2位值 | 颜色 | RGB值 | PHOT渲染色 |
|-------|------|-------|-----------|
| 00 | 无光(黑) | #000000 | 暗 |
| 01 | 红 | #FF0000 | 红色光子 |
| 10 | 绿 | #00FF00 | 绿色光子 |
| 11 | 蓝 | #0000FF | 蓝色光子 |

**注意**：多个组同时设置时，颜色叠加混合。例如组0=01(红)+组1=10(绿)=黄光。

### 常见波长值

| 波长值(hex) | 效果 | 颜色 |
|------------|------|------|
| 0x000000 | 全黑(无光子) | 黑 |
| 0x555555 | 全红(12组全红) | 纯红 |
| 0xAAAAAA | 全绿(12组全绿) | 纯绿 |
| 0xFFFFFF | 全蓝(12组全蓝) | 纯蓝 |
| 0x0000FF | 组0-3蓝,其他无 | 暗蓝 |
| 0xFF0000 | 组8-11红,其他无 | 红偏 |
| 0x00FF00 | 中间组绿 | 中绿 |
| 0xFFFF55 | 蓝+红=紫 | 紫色 |
| 0xFFFFAA | 绿+蓝=青 | 青色 |
| 0x5555AA | 红+绿=黄 | 黄色 |
| 0xFFFFFF | 白(全色谱) | 白色 |

## FILT滤镜实战设计

### 设计1：固定颜色滤波器

```
配置：
Tmp=0(赋值模式)
Ctype=目标波长值

效果：任何光子穿过FILT后变为固定颜色

示例：
┌──────────────────┐
│ 白光IN           │
│ ▼                │
│ [FILT] Ctype=红  │
│ ▼                │
│ 红光OUT          │
└──────────────────┘

应用：色彩校正、信号标识
```

### 设计2：带通滤波器

```
配置：
Tmp=1(AND模式)
Ctype=白名单波长掩码

效果：仅保留与掩码匹配的波长分量

示例：Ctype=0x555555(全红掩码)
┌──────────────────┐
│ 白光IN           │
│ ▼                │
│ [FILT] AND 红色掩码 │
│ ▼                │
│ 仅红光OUT         │
└──────────────────┘

应用：颜色分离、频道过滤
```

### 设计3：红移滤镜链（模拟深空效果）

```
多级FILT串联：
[PHOT源] → [FILT Tmp=4 Temp=173K] → [FILT Tmp=4 Temp=173K] →
→ [FILT Tmp=4 Temp=173K] → 红移后的PHOT

效果：累计红移→光子从蓝→绿→红→红外(不可见)
应用：模拟宇宙学红移、距离指示器
```

### 设计4：颜色合成器（三原色混合）

```
         [红PHOT源] ──▶ [FILT OR 红] ──┐
         [绿PHOT源] ──▶ [FILT OR 绿] ──┼──▶ 合成光
         [蓝PHOT源] ──▶ [FILT OR 蓝] ──┘

每个分支用Tmp=2(OR模式)注入对应颜色分量
```

### 设计5：颜色检测器

```
[待测PHOT] ──▶ [FILT Tmp=1 AND 掩码] ──▶ 检测输出

如果结果!=0 → 含该波长
如果结果==0 → 不含该波长

配合电信号：PHOT→PSCN→电信号→指示器
```

### 设计6：温度驱动色移器

```
可变温度源(HEAC/COOL) → FILT(Tmp=4或5, 红移/蓝移)
                     │
                     ▼
              FILT温度变化 → 色移量变化

应用：温度→颜色转换器（可视化温度）
```

## WIFI频道温度对照（完整）

### 频道计算公式

```
channel = clamp((temp - 73.15) / 100 + 1, 0, CHANNELS-1)
```

### WIFI频道配置建议

- **同频道通信**：确保WIFI粒子温度在相同频道范围内
- **频道隔离**：不同频道的WIFI不互相干扰
- **频道切换**：通过改变温度来切换通信频道
- **频道冲突**：同频道WIFI信号会在同一网络中广播

## FILT的进阶用途

### 与GOL/LIFE配合
将GOL图案产生的"颜色"通过FILT转换为光信号：
```
LIFE区域 ──颜色──▶ FILT ──转换──▶ PHOT(带颜色信息)──▶ 光路传输
```

### 与WIFI配合
FILT可作为WIFI信号的"颜色调制器"：
```
WIFI信号 ──触发──▶ 光子生成 ──▶ FILT滤波 ──▶ 特定波长光信号
```

### 光逻辑门
多个FILT组合可实现布尔逻辑：
```
A光 ──▶ FILT AND ──▶ 结果
B光 ──▶           │
           (AND=仅两者都有对应波长时输出)
```

## Lua操纵FILT示例

```lua
-- 设置FILT模式
local function setFiltMode(filtId, mode, ctype)
    sim.partProperty(filtId, 9, mode)   -- tmp=模式
    sim.partProperty(filtId, 2, ctype)  -- ctype=波长/掩码
end

-- 批量设置FILT为红色透过
for i in sim.parts() do
    if sim.partProperty(i, 0) == elements.getByName("FILT") then
        setFiltMode(i, 1, 0x555555)  -- AND模式,仅红色
    end
end

-- 创建带通滤波器链
function createBandpassFilterChain(x, y, wavelength, temp)
    local id = sim.partCreate(-1, x, y, "FILT")
    if id > 0 then
        sim.partProperty(id, 9, 1)       -- AND模式
        sim.partProperty(id, 2, wavelength) -- 掩码
        sim.partProperty(id, 7, temp or 295.15) -- 温度
    end
    return id
end

-- 颜色检测函数
function detectPhotonColor(photonId)
    local wl = sim.partProperty(photonId, 2)  -- ctype=波长
    local hasRed = bit.band(wl, 0x555555) ~= 0
    local hasGreen = bit.band(wl, 0xAAAAAA) ~= 0
    local hasBlue = bit.band(wl, 0xFFFFFF) ~= 0
    return hasRed, hasGreen, hasBlue
end
```
