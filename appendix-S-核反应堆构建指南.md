## 1. 核能在TPT中的意义

### 为什么要在TPT中建造核反应堆？

TPT是一个基于粒子物理的2D沙盒模拟器。建造核反应堆不仅仅是"为了好玩"，它在TPT的工程生态中占有核心地位：

**驱动大型电路系统** — 裂变/聚变反应堆的持续能量输出可以驱动数百个SPRK逻辑门、PIPE粒子管道网络、PSTN/FRME机械臂阵列等超大规模工程。相比于BTRY(电池)的有限供电范围（菱形4格），蒸汽轮机驱动的PSCN/NSCN太阳能电池板阵列可以跨屏幕输电。

**为粒子加速器供能** — PROT质子碰撞合成元素链（PROT+PROT→NBLE→CO2→O2→PLSM→POLO→URAN→PLUT→SING）需要巨大的初始能量输入。核反应堆是最理想的能源。

**支持极端环境实验** — 许多TPT高端实验（如EXOT→WARP转化、VIBR爆炸链、SING黑洞生成）需要稳定供应的高温高压环境，核反应堆是最佳选择。

**教育与展示** — TPT核反应堆从原理上模拟了真实核工程的核心概念：临界质量、链式反应、中子慢化、聚变约束、SCRAM紧急停堆、纵深防御等，是物理教学和公众科普的优秀工具。

### 核反应堆可以驱动什么？

| 负载类型 | 所需能量形式 | 推荐反应堆类型 |
|---------|------------|-------------|
| 电路系统(SPRK) | 持续SPRK电脉冲 | PHOT光子堆 + PSCN/NSCN光伏板 |
| 蒸汽机械(活塞) | 高压蒸汽(WTRV) | PLUT裂变堆 + WATR锅炉 |
| 粒子管道(PIPE) | SPRK信号 | 任意堆 + 光伏转换 |
| 粒子加速器(PROT) | 高温等离子体 | DEUT聚变堆 |
| 加热系统(HEAT) | 直接热量 | 混合粉末热堆 |
| 激光阵列(PHOT) | 光子束 | ISOZ/ISZS光子堆 |
| 中子源(NEUT) | 中子束流 | PLUT裂变堆 |

## 2. TPT核物理基础

在动手建造反应堆之前，必须理解五个核心物理机制。这些机制直接决定了反应堆的设计、控制和安全边界。

### 2.1 PLUT自发裂变 — 压力加速衰变

```
PLUT → NEUT │ 概率 = chance(1, 100) × chance(5×P, 1000)
```

PLUT(钚)在每一帧有独立的两阶段随机判定是否会自发裂变产生一个NEUT(中子)：

- 第一阶段：`chance(1, 100)` — 1%基础概率，每帧独立判定
- 第二阶段：`chance(5×P, 1000)` — 压力P越大，通过概率越高

压力（P, 单位同环境压力pv）是核反应堆中最危险的变量。在常压(0P)下，第二阶段概率为0/1000=0%，自发裂变几乎不会发生。但当压力升至100P时，第二阶段概率为500/1000=50%，整体自发裂变率上升至约0.5%每帧——考虑到TPT每秒约60帧，这意味着一个PLUT粒子在100P下每秒约有30%概率自发裂变。

**实际影响**：反应堆堆芯中PLUT密集堆积时，每次裂变产生10×CFDS压力，压力又加速更多PLUT裂变，形成正反馈循环。这就是为什么PLUT堆必须配备压力释放系统。

### 2.2 NEUT + PLUT链式反应 — 指数增殖

```
NEUT + PLUT → 裂变 │ 概率 = (3 + pv) / 1000
                │ 1/3 → URAN 或 LAVA(PLUT)
                │ 2/3 → 额外NEUT + 10×CFDS 压力
```

这是整个TPT核工程的基础——链式反应。一个中子击中PLUT时：

1. 以`(3+pv)/1000`概率触发裂变（常压下约0.3%，高压下更高）
2. 裂变结果：三分之二概率产生**额外2个NEUT**（比消耗的1个多，净产1个）
3. 同时释放10×CFDS压力

**临界质量概念**：当PLUT密度足够高时，一个初始NEUT产生的2个次生NEUT中至少有一个会再次击中PLUT触发裂变——形成自我维持的链式反应。净中子倍增因子>1时，反应指数增长直到燃料耗尽或结构被压力摧毁。

**关键数字**：
- 每个NEUT寿命约480-959帧（8-16秒@60fps），在此窗口内必须找到下一个PLUT才能维持链式反应
- 每次裂变产压约+10×CFDS，快速积累

### 2.3 DEUT + NEUT聚变 — 压缩度决定爆炸规模

```
NEUT + DEUT → 聚变爆炸 │ 概率 = (3+pv)/1000 + life/100
                     │ 产物：N 个新NEUT，其中 N = DEUT.life / 50
                     │ N 最大 = 340
                     │ 压力 += 6 × CFDS × N
```

DEUT(重水)的聚变是TPT中单次能量释放量最大的反应：

- DEUT.life代表压缩度/密度（初始10，通过ELEC电解或GRVT引力压缩可提高）
- 聚变产出的NEUT数量 = life/50：
  - life=50（微压缩）→ 1个NEUT（不起眼）
  - life=5000（高压缩）→ 100个NEUT（可观的爆炸）
  - life=17000（理论极大）→ 340个NEUT（灾难性爆炸）
- 触发概率随life增加：基础(3+pv)/1000 加上 life/100，高压缩DEUT几乎必被触发

**DEUT压缩方法**：
- **ELEC电解浓缩**：ELEC+DEUT→DEUT.life+1,temp=0，逐步提高
- **GRVT引力压缩**：增大DEUT.maxlife上限
- **热膨胀规律**：maxlife = 10000/(temp+1)-1，温度越高压缩上限越低
- **扩散平衡**：相邻DEUT交换一半life，高浓度扩散到低浓度

### 2.4 URAN压力-温度反馈

```
URAN: temp = temp × (1 + pv/2000) + MIN_TEMP
```

URAN(铀)不能直接与NEUT反应（只能作为PLUT裂变的1/3概率副产物）。但它有一个独特且危险的性质：**压力指数升温**。

温度每帧乘以(1+pv/2000)因子——压力每增加2000P，温度就翻倍。这是一种非线性正反馈：
- 低压稳定（<100P增长可忽略）
- 中压升温明显（1000P时温度每帧+50%）
- 高压下温度急剧攀升至点燃阈值（>773K引发周围可燃物）

URAN在反应堆中通常作为"被动热源"或裂变副产物收集。

### 2.5 PHOT + ISOZ/ISZS光子链式反应

```
PHOT + ISOZ/ISZS → 额外PHOT │ 概率 = 1/400
                           │ 消耗压力 = -15 × CFDS
```

ISOZ(同位素Z，液态)和ISZS(固态同位素Z)是"光子放大器"——PHOT通过时有1/400概率倍增为2个PHOT：

- 光子数指数增长（1→2→4→8→...）
- 每次倍增消耗局部压力(-15×CFDS)，维持负压环境
- ISZS熔点仅27°C，光子本身初始922°C，可能导致ISOZ熔化回ISOZ

**ISOZ/ISZS自发衰变**：
```
ISOZ/ISZS → PHOT │ 概率 = chance(1, 200) × chance(-4×P, 1000)
```
衰变率与负压正相关——真空环境下(负压)衰变更快。

**光伏效应**：PHOT撞击PSCN/NSCN时直接产生SPRK电脉冲，这是光子堆发电的基本原理。

### 4.3 分步建造

**第一步：铺设结构基础**

```
1. 在地面上平铺3层ROCK地板（抗压120P）
2. 左右两侧竖立ROCK墙，高度20-30像素
3. 顶部留出蒸汽出口通道（宽3-5像素）
```

**第二步：建造中子屏蔽层**

```
1. 在ROCK内壁铺设TTAN层（至少5-8层）
   ┌─ROCK────────────ROCK─┐
   │TTANTTANTTANTTANTTAN  │
   │TTANTTANTTANTTANTTAN  │  ← 8层TTAN衰减中子至~66%
   │TTANTTANTTANTTANTTAN  │
   │TTANTTANTTANTTANTTAN  │
   │TTANTTANTTANTTANTTAN  │
   └──────────────────────┘

2. 底部加厚TTAN至10层（中子向下散射较多）
3. 留出左侧SCRAM注入口（2×2空洞，暂用TTAN临时填充）
4. 留出右侧NEUT点火通道（1×2空洞，用CRMC填充作窗口）
```

**第三步：注入慢化/冷却水**

```
1. 在TTAN内壁内注入WATR，深度约10-15像素
2. 确保WATR连续分布，无空洞
3. WATR同时担任：
   - 中子慢化剂（v×0.995/层，减速快中子）
   - 冷却剂（吸收堆芯热量→沸腾为WTRV）
   - 工质（WTRV压力驱动涡轮）
```

**第四步：放置燃料**

```
1. 在WATR包围的中心区域放置PLUT
2. 布局建议：
   ┌───────────────┐
   │ PLUT PLUT PLUT│
   │ PLUT PLUT PLUT│  ← 2×4或3×3密集排列
   │ PLUT PLUT PLUT│     保持粒子间距≤3像素
   └───────────────┘     确保NEUT能在life耗尽前找到下一个PLUT

3. PLUT上方放置HEAC（导热金属）形成热桥
4. HEAC连接至TSNS温度传感器
```

**第五步：安装传感器与控制电路**

```
控制电路布局（堆芯右侧）：

   BTRY ──→ METL ──→ TSNS(堆芯上) ──→ PSNS(堆芯内)
                          │
                          ├──→ DLAY(红色→报警灯)
                          │
                          └──→ SWCH(SCRAM触发,温度>500°C时闭合)


SCRAM执行电路：

   SWCH(PSCN触发) ──→ 打开LN2注入口
                   ──→ 关闭NEUT注入窗(CRMC→TTAN替换)
                   ──→ 开启真空泵(PUMP,降负压)
```

**第六步：蒸汽出口与涡轮**

```
1. 顶部WATR面上方留3-5像素高空间作蒸汽收集区
2. 蒸汽收集区上方连接PIPE管道→输送WTRV
3. 管道终端面向PSTN活塞或直接喷向WTRV→压力驱动机构
4. 涡轮设计见第9章"能量提取系统"
```

### 4.4 启动程序

```
启动前检查：
[ ] 所有TTAN屏蔽层完整，无裂缝
[ ] WATR冷却剂注满，液位距顶部2-5像素
[ ] 传感器与控制电路通电正常（BTRY工作）
[ ] SCRAM系统待命（LN2储罐就位）
[ ] 真空区/排气通道畅通

点火步骤：
1. 确认PSNS压力读数≈0P
2. 通过CRMC窗口向堆芯注入1个NEUT
   方法A：在CRMC外侧放置1个POLO，待其自发衰变→NEUT穿透CRMC
   方法B：用PLUT替代——在CRMC外用TSNS加热PLUT至自发裂变
3. 观察TSNS温度：应在2-5秒内升至60-150°C
4. 观察PSNS压力：稳定在10-50P为正常
5. 观察WATR：上部水层开始出现WTRV气泡
6. 如果5秒内无温度变化→再注入1个NEUT
```

### 4.5 正常运行参数

| 参数 | 正常范围 | 警戒值 | 危险值 |
|------|---------|--------|--------|
| 堆芯温度 | 60-500°C | 500-800°C | >800°C |
| 堆芯压力 | 5-80P | 80-150P | >150P |
| WATR温度 | 60-99°C | 100°C(沸腾) | >105°C(蒸汽化) |
| 中子通量 | NEUT可见但不密集 | NEUT密集闪烁 | NEUT充满水层 |
| TTAN外温度 | 22-50°C | 50-80°C | >80°C |

### 4.6 SCRAM紧急停堆程序

```
触发条件（任一满足即执行）：
  - PSNS检测压力 > 150P
  - TSNS检测温度 > 800°C
  - TTAN外层温度 > 80°C（中子泄漏）
  - WATR完全蒸发

执行步骤（按顺序，不可跳步）：

□ 步骤1：切断中子供应
   用TTAN替换CRMC窗口，封锁NEUT注入通道

□ 步骤2：注入紧急冷却剂
   打开SWCH→释放LN2进入堆芯区域
   LN2接触→温度骤降至-196°C→PLUT裂变概率趋近0
   WATR残留可能冻结为ICE，但优先降温

□ 步骤3：压力释放
   移除顶部ROCK盖板或开启Vent释放压力
   注意：快速减压可能导致WATR暴沸（闪蒸）
   改用渐进式释压：先开小口，待稳定后再全开

□ 步骤4：燃料隔离
   用DMND或TTAN墙将PLUT分离为小块
   破坏临界质量条件→链式反应自然停止
   方法：PSTN推入DMND隔板分割堆芯

□ 步骤5：长期冷却
   HEAC散热片持续导出余热
   CHLL/FRZZ环境降温
   维持至少30秒（1800帧）直到TSNS<50°C
```

### 5.6 启动与引爆程序

```
引爆前检查：
[ ] DMND容器完整，无裂缝（3层检查）
[ ] DEUT液池稳定，无泄漏
[ ] NEUT注入通道对准
[ ] 压力释放阀打开（WTRV出口畅通）
[ ] 所有人员/无关粒子撤离到TTAN屏蔽层外
[ ] ELEC浓缩系统已运行至目标life值
[ ] GRVT（如果使用）已就位

引爆程序：
1. 记录当前DEUT.life值（通过LSNS或目视颜色）
   life<50：微光引爆（安全，产出0-1个NEUT）
   life=50-500：小规模引爆（产出1-10个NEUT，可控测试）
   life=500-5000：中等聚变（产出10-100个NEUT，需DMND容器）
   life>5000：大规模聚变（产出>100个NEUT，极限操作！）

2. 确认WTRV蒸汽出口对准涡轮/活塞
3. 打开SWCH（点火开关）→PLUT加热→NEUT射入DEUT
4. 引爆瞬间：
   - DEUT池爆发出大量NEUT（白色闪烁粒子）
   - 压力急剧上升（6×CFDS×N 的总压）
   - WTRV（如有水接触）瞬间爆发驱动涡轮
   - DMND容器承受压力冲击

5. 引爆后：
   - NEUT迅速衰减（life 480-959帧）
   - 压力通过WTRV出口释放
   - 清理残留：重新注入新DEUT，重置ELEC系统
```

### 5.8 紧急熄火(SCRAM)

```
DEUT聚变堆SCRAM与PLUT裂变堆不同——聚变是一次性的，没有链式反应可"终止"。
SCRAM的重点是"制止即将发生的引爆"和"减轻引爆后的后果"。

引爆前制止（有预警时）：
□ 1. 切断ELEC电源→停止DEUT.life增长
□ 2. 注入WATR稀释DEUT→降低密度(life扩散平衡)
□ 3. 加热DEUT→热膨胀降低maxlife（但注意：触及其他阈值!）
□ 4. 关闭NEUT注入SWCH→阻止点火信号
□ 5. 用TTAN封堵CRMC窗口

引爆后减轻（引爆已发生）：
□ 1. LN2注满DMND腔内→吸收残余热量
□ 2. WATR淹没→慢化残余NEUT
□ 3. DMND墙完整性检查→如有裂缝，外层TTAN隔离
□ 4. 等待NEUT自然衰减（约8-16秒）
□ 5. 检查放射污染→用TTAN覆盖PLUT/URAN残留
```

## 6. 反应堆C：ISOZ/ISZS光子反应堆

### 6.1 设计原理

光子堆是TPT中最安全、最可控的反应堆类型。它利用ISOZ/ISZS对PHOT的倍增效应（1/400概率产生额外PHOT）和自发衰变（产PHOT）构建光子放大链路。最终通过光伏效应（PHOT→PSCN/NSCN→SPRK）将光能转换为电能。

**核心优势**：
- 无NEUT参与→无放射性污染扩散
- PHOT仅在直线传播→容易控制和引导
- 无临界质量概念→堆芯规模可以任意
- 光子不破坏固体结构（除高温引燃外）

**核心链**：
```
ISOZ/ISZS 自发衰变 → 种子PHOT → 更多ISOZ/ISZS → PHOT倍增(1/400)
→ 大量PHOT → PSCN/NSCN光伏板 → SPRK电流
```

### 6.2 材料清单

| 固态同位素Z | ISZS | 20-40粒子 | 次级增益介质(固态更稳定) |
| 光子 | PHOT | 1个 | 种子光子 |

```

[ ] INSL腔体完整无裂缝
[ ] ISZS层稳定固态（温度<-113°C）
[ ] ISOZ层液体状态（温度>27°C）
[ ] PSCN/NSCN光伏板就位，连线正确
[ ] GLOW种子光源就位

启动步骤：
1. 打开SWCH→BTRY给GLOW通电→GLOW开始产生PHOT
2. 观察PHOT进入ISOZ/ISZS层→出现光子倍增
3. 5-10秒后PHOT数量达到稳态（移出率=倍增率）
4. 检查光伏板：PSCN上出现SPRK闪烁→电流汇入METL总线
5. 用TSNS监测ISOZ层温度：温度应稳定在20-30°C
6. 如果PHOT数量不足→增加GLOW数量或提高GLOW温度

正常稳态指标：
- ISOZ温度：20-30°C
- PHOT密度：充满ISOZ层但不溢出INSL
- SPRK频率：每1-3帧有1个SPRK产生（取决于光伏板面积）
- 系统压力：0~-30P（ISOZ光子倍增消耗压力）
```

### 6.7 紧急关机

```
光子堆SCRAM（最简单最安全）：

1. 关闭GLOW电源（SWCH OFF）→停止种子PHOT产生

```

## 7. 反应堆D：混合粉末热反应堆

### 7.1 设计原理

最简单的入门级反应堆。利用化学反应产热，无核风险，适合新手熟悉热管理系统。

**核心反应**：
```
ACID(酸) + DUST(粉尘) → 中和产热（持续升温，不产生新物质但释放热量）
```

其他可选热源组合：
- COAL/BCOL + FIRE → 持续燃烧产热
- GUNP + SPRK → 定向爆炸产热+压力
- BTRY + DLAY(高T) → 电阻加热（纯电路热）

### 7.2 材料清单

| 材料 | 代码 | 数量 | 用途 |
|------|------|------|------|
| 酸 | ACID | 50-100粒子 | 热源反应物A |
| 粉尘 | DUST | 50-100粒子 | 热源反应物B |
| 水 | WATR | 大量 | 热交换工质 |
| 导热金属 | HEAC | 约30 | 热桥→WATR |
| 钻石 | DMND | 约50 | 耐酸容器壁 |
| 金 | GOLD | 可选 | 耐酸二级容器 |
| 温度传感器 | TSNS | 2 | 温度监测 |
| 绝缘体 | INSL | 约20 | 外部热隔离 |
| 蒸汽管道 | PIPE | 约20 | WTRV输出 |
| 活塞 | PSTN | 2-4 | 机械功提取 |

### 7.4 分步建造

**第一步：耐酸容器**

```
1. 用DMND建造容器（ACID腐蚀大多数金属但DMND免疫）：

   俯视：
   D D D D D D D D D D D D
   D                       D
   D                       D
   D                       D
   D D D D D D D D D D D D

2. DMND壁厚至少2层
3. 替代方案：GOLD（通电时抗酸）或ROCK（便宜但不美观）
```

**第二步：填充反应物**

```
1. 容器底部先放DUST（粉尘）50-100粒子
2. DUST上方放ACID（酸）50-100粒子
3. ACID自动下沉与DUST接触→反应产热
4. 反应不消耗粒子本身，持续产热
```

**第三步：热交换系统**

```
1. 在反应区上方铺设HEAC（导热金属）层：
   ┌──────────────────┐
   │ H H H H H H H H  │ ← HEAC导热桥
   │ H H H H H H H H  │
   └──────────────────┘

2. HEAC上方注入WATR水层（深5-10像素）
3. HEAC将反应区热量传导给WATR→WATR升温→沸腾→WTRV蒸汽
4. WTRV上升至蒸汽收集空间→通过PIPE导出至涡轮
```

**第四步：控制系统**

```
反馈控制回路：

   TSNS(反应区) → DLAY → 比较器 →
   ├── 温度<100°C → 打开ACID供应阀（增加反应强度）
   ├── 温度=200-300°C → 正常运行
   └── 温度>400°C → 打开LN2冷却阀或关闭ACID供应

液位控制：

   TSNS(WATR层, 检测蒸干) → 自动补水阀(PIPE+SPRK)
```

### 7.5 启动程序

```
启动前检查：
[ ] DMND容器完整
[ ] DUST/ACID分别就位
[ ] HEAC热桥完整
[ ] WATR液位正常
[ ] PIPE蒸汽管道畅通
[ ] TSNS/PSNS传感器工作

启动步骤：
1. DUST+ACID接触即自动反应（无需额外点火）
2. 10-20秒内反应区温度升至100-300°C
3. WATR层底部开始出现WTRV气泡
4. WTRV通过PIPE导出→驱动涡轮
5. 调整ACID/DUST比例控制反应速度：
   - 更多ACID→反应更快
   - 更多DUST→反应更均匀
6. 稳定运行温度维持200-300°C

SCRAM停机：
1. 用PSTN推入分离墙→隔离ACID和DUST
2. 或注入BASE（碱）中和ACID
3. LN2降温→所有反应降至低温
4. 关闭PIPE蒸汽阀
```

### 8.1 中子屏蔽材料对比

```
│   └─────────────────────────────────────────────────┘       │
└─────────────────────────────────────────────────────────────┘
```

**蒸汽轮机设计要点**：
- WATR锅炉离堆芯越近效率越高，但越容易过热蒸干
- PIPE管道方向对齐蒸汽流向（WTRV自然上升）
- PSTN活塞需要弹簧复位机制（FRME框架或反向PSTN）
- 压力积累在封闭管道中可达到极高——必须有泄压阀

## 10. 常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| PLUT堆链式反应无法启动 | NEUT注入数量不足或PLUT密度不够 | 增加PLUT密度至3×3以上；增加注入NEUT至2-3个；提高局部压力辅助裂变概率 |
| 链式反应失控(温度/压力飙升) | 临界质量过大,中子正反馈过强 | 减少堆芯PLUT数量；增加TTAN吸收层介入；立即触发SCRAM程序 |
| PLUT堆压力突然归零 | 蒸汽通道过大或压力壳破裂 | 检查ROCK/INSUL完整性；检查顶部安全膜是否过早碎裂；缩小蒸汽出口尺寸 |
| WATR冷却剂完全蒸发 | 堆芯温度过高或WATR量不足 | 增加WATR初始量；降低堆芯PLUT密度；安装自动补水系统 |
| DEUT聚变产出NEUT太少 | DEUT.life过低(未充分浓缩) | 用ELEC电解浓缩DEUT.life至50以上；添加GRVT引力压缩 |
| DEUT聚变产出NEUT巨大失控 | DEUT.life过高(过度压缩) | 控制ELEC电解时间；监测DEUT颜色(life≥240发蓝白光)→预警阈值 |
| DEUT容器(DMND)内部爆炸 | life>10000的DEUT被NEUT触发,产出>200个NEUT | 永远不要将DEUT.life提升至5000以上除非明确知道后果 |
| PHOT堆光子数不够 | ISOZ/ISZS增益介质不足；或种子PHOT太少 | 增加ISOZ/ISZS层厚度；增加GLOW数量；检查ISOZ是否已熔化(温度>27°C) |
| PHOT堆光伏板无输出 | PSCN/NSCN布局错误；或PHOT未达到光伏板 | 确保PSCN面朝PHOT来向；检查NSCN在PSCN后方；检查PSCN→METL连线 |
| ISZS熔化(固态层消失) | 光子温度(922°C)加热ISZS超过27°C熔点 | 用COOL持续冷却ISZS层；在ISZS和ISOZ之间加导热间隔层；降低PHOT通量 |
| TTAN外仍有NEUT泄漏 | 屏蔽层太薄或层间有间隙 | 加厚TTAN至10-20层；检查是否有CRMC窗口以外开口；用GOLD替换部分TTAN提高衰减 |
| 中子意外嬗变周围材料 | NEUT散射路径上存在可嬗变材料 | 检查NEUT可能传播的所有路径；移除WATR→DSTW路径上的可燃物；添加TTAN挡板 |
| SCRAM系统无法触发 | 传感器或控制电路断开 | 定期测试SCRAM电路；BTRY确保供电；使用冗余传感器（2个TSNS+2个PSNS） |
| 压力释放阀卡死 | ROCK碎片堵塞或PUMP被熔毁 | 使用DMND释放阀；安装多个冗余压力释放通道(至少2个)；被动安全膜+主动PUMP双保险 |
| ACID+DUST热堆温度上不去 | 反应物比例不当或接触面积不够 | 确保ACID能接触到DUST；增加HEAC热桥面积；减少WATR水量(减少热容) |
| 电路被EMP摧毁 | SING爆发或其它EMP源 | 关键控制电路用BMTL/BRMT保护；使用DMND隔离关键SWCH；冗余备份电路 |

## 12. 核反应堆安全操作检查表

### 12.1 建造前安全规划

- [ ] 选定反应堆类型(PLUT/DEUT/PHOT/热堆)，确认有足够建造空间（建议至少80×60像素区域）
- [ ] 准备所有材料，确认数量充足
- [ ] 设计安全壳层级（至少3层：慢化+吸收+压力壳）
- [ ] 规划SCRAM紧急停堆系统（不能事后追加！）
- [ ] 规划紧急冷却系统（LN2/FRZZ储罐就位）
- [ ] 确认NEUT传播路径上无意外可燃物(PLNT/WOOD/COAL/FIRE)
- [ ] 确认控制电路电源独立于反应堆输出（BTRY单独供电）
- [ ] 设计被动安全特性（安全膜、泄压阀）——不能依赖主动控制

### 12.2 建造中检查

- [ ] 所有墙类(WL_*)连续无裂缝（用Debug模式H键逐像素检查）
- [ ] TTAN屏蔽层无间隙（NEUT可从1像素缝隙穿出）
- [ ] CRMC观察窗之外无其他NEUT透明通道
- [ ] DMND容器壁≥3层（如果使用DMND）
- [ ] 传感器(TSNS/PSNS)紧贴被监测区域
- [ ] 控制电路连线无断路/短路
- [ ] 所有SWCH默认在安全位置(OFF状态)
- [ ] 管道(PIPE)方向正确，颜色循环无断点
- [ ] WATR冷却剂液位标识清晰（知道何时该补水）

### 12.3 启动前最终检查

- [ ] 人员(火柴人STKM)撤离到TTAN屏蔽层外
- [ ] 所有SCRAM子系统单独测试通过：
  - [ ] LN2注入阀能正常开启
  - [ ] NEUT阻断器(TTAN)能就位
  - [ ] 压力释放阀(PUMP/VENT)能启动
  - [ ] 燃料隔离器(PSTN+DMND)能动作
- [ ] 温度传感器校准（与已知参考温度对比）
- [ ] 压力传感器归零（正常大气压）
- [ ] 备份BTRY电力充足
- [ ] 录制存档快照（以防事故后难以还原现场）
- [ ] 准备手动干预工具（PROP工具在工具栏就绪）

### 12.4 运行中监控

- [ ] TSNS读数在正常范围（参见各反应堆正常参数表）
- [ ] PSNS读数在正常范围
- [ ] TTAN外层温度无异常升高（指示中子泄漏）
- [ ] WATR液位不低于警戒线（不露出堆芯）
- [ ] 蒸汽管道WTRV流动顺畅（无堵塞/冷凝）
- [ ] 不依赖单一传感器——至少2个独立TSNS/PSNS互相校验
- [ ] 定期检查SCRAM系统是否仍可触发（不被熔毁/卡死）

### 12.5 紧急情况应对（按优先级）

```
优先级1 — 立即行动（生命/结构安全）:
□ PSNS >200P 或 TSNS >900°C → 全系统SCRAM
□ DMND容器出现裂缝 → SCRAM + TTAN包围隔离
□ NEUT大规模泄漏到开放区域 → SCRAM + 全局WATR淹没

优先级2 — 紧急处置（防止恶化）:
□ PSNS 150-200P 或 TSNS 600-900°C → SCRAM或大幅降功率
□ WATR完全蒸发 → 紧急补水或SCRAM
□ 控制电路断电 → 手动触发SCRAM（用PROP工具直接修改SWCH状态）

优先级3 — 预警响应（调整运行参数）:
□ TSNS(外层) >80°C → 检查屏蔽层,降低功率
□ NEUT单粒子逸出 → 加厚TTAN局部区域
□ WTRV流量下降 → 检查管道堵塞或WATR不足
```

### 12.6 停机后检查

- [ ] 确认所有NEUT已衰减消失（约等待20-30秒）
- [ ] TSNS读数降至<50°C
- [ ] PSNS读数归零或接近零
- [ ] 检查结构完整性：所有墙类无裂缝
- [ ] 检查燃料状态：PLUT是否已消耗/转为URAN/LAVA
- [ ] 检查放射性污染：TTAN外是否有意外PLUT/URAN粒子
- [ ] 清理残留：移除LAVA、凝固的URAN等副产物
- [ ] 补充消耗品：WATR/LN2/TTAN
- [ ] 记录事故/异常日志（用于改进设计）
