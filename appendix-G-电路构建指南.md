# 附录G：电路构建指南

## 基础概念

TPT的电路系统基于SPRK电脉冲传导。SPRK不能独立存在，必须放在导体上。

### 传导距离

- 正常导体间距≤1像素时电脉冲可跳过空隙传导
- 两导体中点有INSL/RSSS时阻断
- INST一帧内传导整条线路(FloodINST)
- GOLD十字方向4格远程导电
- WIFI无线跨空间传导(同频道)

### SPRK生命周期

| 导体 | life帧数 | 说明 |
|------|---------|------|
| METL/PSCN/NSCN等 | 4帧 | 前4帧激发，后4帧抑制 |
| WATR | 6帧 | 水中较慢 |
| SLTW | 5帧 | 更快导电 |
| SWCH(ON) | 14帧 | 开关通电时长 |
| RSST | 5帧 | 耗尽后自毁 |

## 基本电路模式

### 二极管
```
PSCN → [任意导体] → NSCN
```
PSCN不向NSCN输出，NSCN可向PSCN输出。电流只能从PSCN流向NSCN。

### 太阳能电池板
```
PHOT → PSCN(上)+NSCN(下)紧贴 → SPRK
```
光子撞击P型硅时直接产生SPRK。

### 开关(SWCH)
```
PSCN → SWCH → 开启(life=10)
NSCN → SWCH → 关闭(life=9)
```
SWCH ON时作为sender可向金属导电，不向PSCN/NSCN/WATR等导电。

### 非门(NOT门)
```
输入 → METL → PTCT(<100℃时导电) → 输出
       METL → PTCT加热至200℃(导电关闭)
```
或使用NTCT(>100℃时导电)。

### 与门(AND门)
两个NTCT并联：两个输入都加热各自NTCT>100℃时，输出端才同时导通。

### 延时器
```
PSCN → DLAY → (等待temp-273.15帧) → NSCN输出
```
默认4帧延迟。用HEAT/COOL调整温度改变延迟时间。

## 高级电路元件

### 特斯拉线圈(TESC)
SPRK在TESC上时生成LIGH闪电。Tmp控制强度(上限300)。大Tmp=频率低但闪电强，小Tmp=频率高但闪电弱。

### 电极(ETRD)
两个ETRD，一个通电→两点间PLSM连线(9000℃+)。INSL阻隔则无效。

### WiFi无线
同频道WiFi间无线通信。频道=(temp-73.15)/100+1。每升高100℃换一个频道。101个频道互不干扰。

### 电池(BTRY)
永久电源。菱形范围(距离<4)持续通电。排除WATR/SLTW/NTCT/PTCT/INWR。

## 常见电路模式速查

### 上升沿/下降沿检测
```
信号 → DLAY(延时)→ SWCH → 只在边沿(上升/下降)触发
```
- 上升沿检测：SWCH 在输入信号到来时瞬时导通一帧
- 下降沿检测：信号断开后 SWCH 残留一帧

### SR锁存器(Set-Reset)
```
SET(PSCN) → SWCH_A(life=10, ON) → 输出
RESET(NSCN) → SWCH_A(life=9, OFF)
```
SWCH 保持上一状态——SET 激活打开，RESET 激活关闭。比单个 SWCH 可靠。

### 时钟发生器
```
BTRY → DLAY(高温) → PSCN输出 → 反馈回DLAY → 循环
```
DLAY 温度决定振荡频率。temp=60℃→60 帧周期≈1Hz。METL→DLAY→PSCN→METL→DLAY 形成反馈环。

### 计数器
```
时钟 → STOR(吸收粒子) → STOR计数 → DLAY→ 输出
```
每次脉冲 STOR 吸收一个标记粒子。计数后读出粒子数。

### 七段数码管
每个段 = 独立 SWCH + 导线 + METL 灯丝。7 个段由 7 条独立控制线驱动。使用 BTRY 供电→SWCH→导线发光显示数字。

### PWM(脉宽调制)
```
BTRY → DLAY(高温=长周期) → SWCH → 输出
HEAT/COOL 调整占空比
```
DLAY 高温=长导通，低温=短导通。用于模拟量控制(如 PUMP 压力调节)。

### 顺序控制器
```
DLAY_A → DLAY_B → DLAY_C → ... → DLAY_n
```
链式延时器，每个触发下一个。实现时序控制和分步自动化。

## 实用电路技巧

1. **LED 显示**：通电 METL/PSCN/NSCN 因自加热(10℃/传导)逐渐变热发红→高温发光(PROP_HOT_GLOW)。长导线就是灯丝。
2. **触摸开关**：利用 METL→SPRK→METL 传导链实现"粒子接触即通电"。
3. **温度编码**：TSNS 检测不同温度→SPRK→不同功能。HEAT/COOL 远程调节 TSNS 阈值。
4. **压力逻辑**：PSNS 检测压力阈值→SPRK。结合 PUMP 实现压力反馈控制。
5. **生命检测**：LSNS 检测特定 Life 值→触发。用于状态机控制。
6. **多路复用**：WiFi 多频道=多条独立通信线。不同频道间完全隔离。
7. **绝缘技巧**：INSL 放在导线间阻断传导但不阻断粒子物理接触。RSSS 阻隔更强。

## 常见电路问题

1. **信号串扰** → 在导线间放 INSL 绝缘
2. **反馈回路** → 使用二极管(PSCN→NSCN 单向)或 SWCH 互锁
3. **延时过长** → 检查 DLAY 温度，冷却可加快
4. **WiFi 不同步** → 检查频道是否一致(温度)，WiFi 有 1 帧延迟
5. **EMP 破坏** → EMP 会概率性摧毁所有带电电子设备——关键电路用 BMTL/BRMT 保护或用 DMND 隔离
6. **热失控** → 导体自加热可能导致热熔毁——TUNG 耐受高温(3422℃)适合高温电路
7. **水位导电** → WATR/SLTW 导电，水浸入电路会导致短路——用墙类(WL_*)隔离水路
