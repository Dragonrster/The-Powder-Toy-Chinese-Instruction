## 多步合成链

### 钢铁制取
```
SAND + COAL → LAVA加热 → SLCN(硅粉) → LAVA(SLCN) + LAVA(METL) → PSCN + NSCN
IRON + COAL/BCOL(加热熔化) → LAVA(IRON) → 倒在COAL上冷却 → METL(金属)
```

### 核燃料循环
```
URAN → (高压加热) → 自发裂变产NEUT → NEUT + PLUT → 裂变(链式反应)
PROT + PROT碰撞(tmp>420) → URAN → 高压升温 → 产NEUT → NEUT + URAN → (需先转PLUT)
POLO + 10 PROT → PLUT → NEUT轰击 → 裂变(产2NEUT+能量)
```

### 奇异物质链
```
BREC → (高温>9000℃+高压>30) → EXOT → tmp2>6000 → WARP → tmp2>2000 → ELEC
EXOT + LAVA(TTAN/GOLD) → LAVA(VIBR) → VIBR爆炸 → EXOT + ELEC + PHOT + BREC
VIBR + ANAR → BVBR → EXOT熔化 → 2xEXOT
```

### 酸→碱→肥皂链
```
WATR + CO2 → CBNW(苏打水)
DSTW + SMKE(双方40~60℃, 1/100) → BASE(碱)
BASE(life≥70) + OIL → SOAP(肥皂)
```

### 光电系统
```
PHOT → PSCN/NSCN(紧贴) → SPRK(太阳能电池)
RSST + PHOT → RSSS(光固化)
GLOW + GEL → RSST(光刻胶制备)
RSSS + NEUT → 液态→还原为ctype物质(可逆光刻)
```

### 制冷循环
```
RFRG(制冷剂) + GAS(pres>3) → CAUS(酸气)
CAUS + WTRV(1/250) → 进一步反应
LN2 < 63K → NICE(氮冰) → >63K → LN2(液氮) → >77K → NONE(蒸发)
```

## 催化反应汇总(PTNM)

PTNM(铂)是 TPT 中的万能催化剂，可催化以下反应：

| 反应物 | 条件 | 产物 | 真实化学对应 |
|--------|------|------|-------------|
| WTRV + BCOL | >200℃, pres>7 | OIL | 费托合成(Fischer-Tropsch) |
| H2 + O2 | — | DSTW×2 | 催化燃烧 |
| GAS | ≥200℃, pres>2 | INSL | 聚合反应 |
| SMKE | — | CO2 | 催化氧化 |
| BREC | >1000℃, pres>50 | EXOT | 奇异物质合成 |
| RSST | — | BIZR | 光刻胶转化 |
